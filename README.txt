The folder CompleteDietManagementApp contains the entire application which can be build in maven (command: mvn clean install) to generate the war and that can be deployed in any server to get the application started. It contains the back-end source code and the compiled front-end code.
The folder FrontEndCodeForReference contains the Angular source code. This has been added for reference.
Diet Management - UI Test Cases.docx and Diet Management DB Design.docx have been added for reference.
